# Exercise 01 — Device Specification Tracker

## Context
Tech product catalog (phones, laptops, tablets)

## Task
Create a class `Device` with the following attributes:
- id
- brand
- model
- price
- releaseYear

Write a program that:
- Creates an array of at least 5 `Device` objects
- Displays all devices
- Finds the newest device
- Finds the most expensive device

## Constraint
- Use only arrays (no ArrayList)
- No static utility methods except `main`

# Exercise 02 — Ride Snapshot Recorder

## Context
Ride-sharing app snapshot (Uber-like, but simplified)

## Task
Create a class `Ride` with the following attributes:
- rideId
- distanceKm
- durationMinutes
- fare

## Program Requirements
- Store multiple `Ride` objects in an array
- Calculate:
  - Average fare
  - Longest ride
  - Cost per kilometer for each ride

# Exercise 03 — IoT Sensor Snapshot System

## Context
Smart building monitoring

## Task
Create a class `SensorReading` with the following attributes:
- sensorId
- location
- value
- unit
- timestamp

## Program Requirements
- Create an array of `SensorReading`
- Display all sensor readings
- Find:
  - Highest reading value
  - Average reading per location
