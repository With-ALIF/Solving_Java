## Factorial Calculation

# Problem Statement
Factorial is a mathematical concept that represents the product of all positive integers up to a given number. It is denoted by an exclamation mark (!). 
For example, the factorial of 5 (written as 5!) is calculated as follows: `5! = 5 × 4 × 3 × 2 × 1 = 120`

# Output

The program should output number `120` when the input is `5`.
